## Deployment Steps

### Step 1: Prepare Local Repository
```bash
# Remove local database (optional, will be regenerated)
del db.sqlite3

# Verify .gitignore is working
git status
# Should NOT show: db.sqlite3, .env, __pycache__, venv/
```

### Step 2: Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit: Hospital Management System"
git remote add origin https://github.com/YOUR_USERNAME/hospital-management-system.git
git branch -M main
git push -u origin main
```

### Step 3: Deploy on Render
1. Go to https://render.com
2. Sign up with GitHub
3. Create new Web Service
4. Select your repository
5. Configure:
   - **Build Command:** `./build.sh`
   - **Start Command:** `gunicorn hospital_system.wsgi:application`
6. Add environment variables (Render will auto-generate SECRET_KEY)
7. Create PostgreSQL database (optional but recommended)
8. Deploy!

### Step 4: Post-Deployment
```bash
# In Render Shell:
python manage.py createsuperuser
python manage.py load_sample_data  # Optional
```
